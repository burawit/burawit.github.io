# burawit.github.io

Simple landing page repository for GitHub Pages.

## About

This repository hosts a basic GitHub Pages site using the repository `master` branch.

## Usage

- Add HTML, CSS, and assets to the repository root.
- Push changes to `master`.
- GitHub Pages will serve the site at `https://burawit.github.io`.

## Notes

- Keep files simple and static.
- Use `index.html` as the landing page.
- Update this README with any project-specific details.
